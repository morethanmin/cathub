## Getting started

### Install

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

### deploy

```bash
# install firebase CLI
$ npm install -g firebase-tools

# login
$ firebase login

# deploy
$ npm run deploy

```
